# Coffeecient — React Native (Expo)

A full React Native conversion of Coffeecient, ready for Expo Go.

## Quick Start

### 1. Install dependencies
```bash
npm install
```

### 2. Start the dev server
```bash
npx expo start
```

### 3. Open in Expo Go
- Install **Expo Go** on your phone (iOS App Store / Google Play)
- Scan the QR code shown in the terminal
- The app will load instantly

## Screens
- **Onboarding** — 3-slide welcome flow (shows every launch)
- **Home** — Hero banner + feature cards
- **Products** — Menu grid with product detail modal
- **About** — Story, stats, and team
- **Login / Register** — Auth with validation
